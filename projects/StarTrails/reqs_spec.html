Hardware and Software requirements

In terms of hardware requirements, we don't have a lot to worry about besides the physical device for the app. All of our requirements can be met using online API calls and a cross-platform mobile application framework. Due to the even spread of Android and IOS users as indicated from our user feedback, our project will need to be developed on software compatible with both Android and Apple operating systems. Which bring us into the software requirements, which as stated before, we will be using a cross-platform mobile application framework called React Native. This allows us to write code using JavaScript, which we are all familiar with, and can deploy to both IOS and Android devices.
  • Hardware: Android, iPhone, Tablet
  • Development: React Native, Flask (alternative)
  • Database/Storage: MongoDB, PouchDB, SQLite, MySQL
  • Others: REST API, Open-Sourced Data
